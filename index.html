Development of friendly interfaces

Interfaces amigáveis e práticas, cores agradáveis e fontes confortáveis aos olhos são a triades de qualquer profissionais que trabalha com projetos visuais. E pensando no Desenvolvimento dessas habilidades aqui estão alguns projetos de UX/UI.

Projetos

Works

Aqui estão alguns projetos desenvolvidos com o intuito de praticar as diferentes linguagens de programação aprendidas.

Here are some projects developed in order to practice the different programming languages learned.

Java

Aplicação para inscrição de residências e cálculo de IPTU. Projeto desenvolvido na disciplina de programação orienta a objeto I.

Application for registration of residences and calculation of IPTU. Project developed in the discipline of programming oriented to object I.

Csharp

O projeto simula, o funcionamento de uma agência bancaria onde há abertura de conta, deposito, saque e transferência de uma conta para outra. Projeto desenvolvido de forma independente

The project simulates the operation of a bank branch where an account is opened, deposits are withdrawn and transfers from one account to another. Project developed independently.

JavaScript

Projeto desenvolvido na disciplina de aplicação web da faculdade. A aplicação visá fazer conversões entre quatro moedas Euro, Dolar, Libra esterlina e Real (EUR, USD, GBP e BRL).

Project developed in the discipline of web application of the college. The application aims to make conversions between four currencies Euro, Dollar, Pound Sterling and Real (EUR, USD, GBP and BRL).

Python

Projeto desenvolvido na disciplina de laboratório I. O projeto visá a analise estátistica de um bando de dados (arquivo.csv).

Project developed in laboratory discipline I. The project aims at the statistical analysis of a database (file.csv).
